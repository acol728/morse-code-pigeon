About Morse Code

Morse Code was developed in the 1830s and 1840s by Samuel Morse, with the first message sent in 1844. It is a language comprised completely of dots and dashes, known as dits (pronounced with a silent “t”) and dahs. While at first used visually, it soon turned into a primarily sound-based language, with dits and dahs represented by short and long beeps.

Rules of Morse Code:

  1. There should be no spaces between parts of a letter.
  2. There should be one space between letters.
  3. A space between words is represented by "/".

Example: "Hello World" in Morse Code is ".... . .-.. .-.. ---/.-- --- .-. .-.. -.."
